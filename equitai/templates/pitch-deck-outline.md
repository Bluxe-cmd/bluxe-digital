### SLIDE 5: HOW IT WORKS

**Headline:** *Five layers. One operating system.*

**Content:** Architecture diagram (simplified)

```
[Community] ←→ [Curriculum Engine] ←→ [Facilitator Network]
                       ↓

                       ↓

```

- Today: CBA money goes to generic commitments with no delivery system

**The model:**
```
Developer signs CBA → Workforce training obligation → EquitAI named as training partner
→ EquitAI delivers cohorts → Graduates get hired → Developer meets CBA commitment
```

**Why this works:** Developer doesn't have to build a training program. Community gets real outcomes. EquitAI has structural, multi-year revenue.

## Design Production Notes

When building the actual deck (Google Slides / PowerPoint / Canva):
1. Use brand palette (Mahogany + Cream + Deep Sage + Gold)
2. Every slide: One headline, one visual, minimal text
3. Font: Cormorant Garamond for headlines; Syne for body
4. Data slides: Use DM Mono for numbers
5. Photos: Real community photos, not stock AI images
6. Animation: Minimal — appear on click only where necessary
7. Aspect ratio: 16:9 widescreen
8. File format: Export as PDF for sharing; keep editable file
